# 投资人研究方法论

> 核心目的：快速建立陌生赛道认知框架，定位方向值不值得深入。

---

## 一、投资判断主线：五问框架

投资决策的主线并不复杂。对任何技术方向，回答五个问题，投资逻辑就出来了。所谓"抓主线"——不是把技术搞透，是用这五问快速定位方向值不值得深入。

| # | 问题 | 判断什么 |
|---|------|----------|
| 1 | **规模**：市场有多大？需求增长多快？谁在付费、为什么付费？ | 天花板够不够高 |
| 2 | **壁垒**：谁在做？追赶周期多长？有没有替代路线？ | 能不能守住 |
| 3 | **稀缺性**：供给跟不上需求的错配窗口有多久？哪个环节最缺？ | 短期有没有超额利润 |
| 4 | **成长性**：这个需求是一次性的还是持续的？范式收敛了吗？ | 值不值得长期投入 |
| 5 | **不确定性**：哪些判断依赖的信息你还没拿到？哪些数字没核实？ | 投资纪律，防止过度自信 |

**用法**：五问套进去，逻辑就出来了。第五问是纪律——强制标注自己不确定的地方，避免信息不全时下结论。

---

### 案例 A：KV Cache 方向

| 问题 | 展开 |
|------|------|
| 规模 | 推理需求增长多快，存储缺口有多大？ |
| 壁垒 | HBM 谁在做，扩产周期多长，有没有替代方案？ |
| 稀缺性 | 短期内存储供给跟不上需求增长，错配窗口有多久？ |
| 成长性 | 这个需求是一次性的还是持续的？ |
| 不确定性 | 替代架构（如线性注意力）进展如何？会不会绕过 KV Cache？ |

### 案例 B：World Model 方向

| 问题 | 展开 |
|------|------|
| 规模 | 商业闭环是什么？谁付费？（自动驾驶仿真？机器人训练？视频生成？） |
| 壁垒 | 各赛道核心壁垒是什么？（标准？数据？算力？场景先占？） |
| 稀缺性 | 供需 gap 在哪？哪个环节缺人/缺钱/缺数据？ |
| 成长性 | 范式收敛了吗？哪条路线正在赢？（VLA vs 具身 RL，像素 vs 潜空间） |
| 不确定性 | World Model 能否直接输出 action？还是只做环境预测？路线尚未收敛。 |

---

**我们的系统要服务的就是这个过程的前半段：把信号收集好、结构化，让投资人能更快跑完这五问的推导。**

## 三、执行方法：四步法（AI 辅助快速建立认知框架）

> 以下方法论来自实战总结：投行人用 AI 工具在 4 小时内完成陌生赛道的拜访准备。
> 核心优势：AI 把"建立认知框架"从两天压到两小时，且不会漏掉维度。

---

### 第一步：搜集信息（约 45 分钟）

工具：Perplexity（注明引用来源，不编信息），配合"横纵分析法"。

**横纵分析法**的核心逻辑——强制从两个维度思考：

- **纵向**：这件事的历史是什么？怎么演变到今天？之前有没有人失败过，为什么？
- **横向**：同一赛道里，别人怎么做的？谁跑在前面，各自策略差异是什么？

两个维度跑完，对一个行业的基本地图就有了。不是全貌，但够做判断、提问题。

### 第二步：整合信息（约 30 分钟）

工具：Claude / Gemini，把搜来的内容 + 研报一起整合，提炼出可以直接开口说的认知框架。

关键 Prompt 技巧：
- 不要问"帮我总结一下这个行业"（出来的是废的）
- 要围绕具体判断问题来问
- **第五条最关键：让 AI 主动告诉你它的不确定性**，防止在客户面前说了编出来的数字

输出标准：200 字能讲清核心逻辑 + 核心竞争力 + 核心挑战 + 可能的合作/投资方向。

### 第三步：写材料（约 1 小时）

让 AI 出初稿，自己改格式和细节。结构：
1. 行业概述（300 字）
2. 主要玩家与竞争格局
3. 目标公司/方向核心情况
4. 潜在投资/合作方向

收到初稿后三件事：核实数字、加自己判断、加团队定位。

### 第四步：可视化（约 1.5 小时）

内容框架让 AI 出，再排版。内部用，简洁直接。

---

### 番外：水下信息问题清单（最被低估，最有价值）

能决定项目走不走得通的信息，一般不在公开资料里。AI 能帮你想到：哪些问题该问，怎么问不冒犯。

两类问题：
- **适合直接问的**：回答能帮你判断投资方向的
- **需要换方式问的**：太直接可能冒犯，换个角度切入

这份问题清单，是见面时的气质、技巧和底气，比 Word/PPT 本身值钱得多。

换电就是把汽车的电池包做成标准化可替换模块，就像换墨盒一样。车开到换电站，把旧电池取出来换一块充满电的，整个过程3-5分钟，比充电快十倍以上。

这个逻辑对重卡特别成立，因为重卡停下来充电就是亏钱——一辆重卡每天跑长途，停两个小时充电，相当于白跑几十公里的运费。换电的时间成本几乎可以忽略，司机当然愿意付费。

商业闭环是这样的：时代电服建站、持有电池，向运输公司收换电服务费，电池资产留在时代电服手上，通过换电次数摊销。矿区、港口这类高密度场景，换电频率高，单站回本相对较快。时代电服的核心竞争优势

第一个，电池供应链。宁德时代专门为换电场景设计的电芯，能承受数千次反复插拔后性能不明显衰减，这个门槛很高。其他做换电的，要么用自研电池，要么从外面采购——成本和一致性都比不上。

第二个，标准话语权。时代电服主导制定了商用车换电电池的国家标准，其他厂商只要做兼容换电，就得跟它对接。这是护城河，不是简单的先发优势，后来者要绕过去代价很高。

第三个，场景先占。矿区是高壁垒市场，进场需要资质、需要和矿方谈判、需要建立供应链关系。时代电服在内蒙古、山西、陕西的矿区已经占了大量位置，后来者复制代价极高。换电行业核心挑战

挑战一：重资产扩张压力。换电站建设成本高，电池也是资产，规模越大资金压力越重，盈利之前需要持续融资。这对时代电服来说既是考验，也是为什么它需要和资本市场打交道。

挑战二：乘用车标准碎片化问题。蔚来做自己的标准，宁德时代做自己的标准，未来乘用车换电要起量的话，标准之争会很复杂。不过这对时代电服目前的商用车业务影响有限。

挑战三：政策依赖度高。换电站建设享受补贴，如果补贴退坡，单站经济模型会有压力，需要通过规模效应来对冲。从投行视角，可能给这家公司提供的服务

A方向：股权融资。时代电服仍在扩张阶段，资金需求持续存在，从A轮到Pre-IPO，窗口可能都有机会。关键是判断它现在处于哪个阶段、是否已有主承销商在跟进。

B方向：战略并购顾问。换电行业可能出现整合，小玩家退出或被收购，时代电服作为领导者可能是买方，可能需要FA服务。

C方向：境外融资。如果时代电服有出海计划——东南亚、中东的矿区或港口——境外债或绿色债可能有需求。

D方向：IPO辅导。时代电服是宁德时代旗下独立运营子公司，未来独立上市是一条很自然的路径。最长线但含金量最高的方向。⚠️ 我不确定的地方

换电站总数（报告里的数字我无法确认是否最新，建议从官方公告核实）；服务换电重卡的保有量（各家统计口径不一）；单站盈利周期和经济模型（属于商业敏感信息，公开资料较少）。

这三个数字，见面的时候直接当问题问对方，既核实了信息，又显得你做过功课。整合这一步收到初稿以后，你还要花大概10分钟改改——主要是把太书面的表达换掉，以及把自己对A/B/C/D方向的判断加进去（AI的排序不一定符合ta对这家客户的具体判断）。总共大概30分钟。第三步：写Word材料（约1小时）这一步最省事，直接让AI出初稿，自己改格式和细节。同样用Claude，Prompt是这样写的：请帮我写一份投行拜访前的行业背景材料。

换电，指以标准化电池包替换作为新能源汽车补能方式，区别于传统插电式充电。该模式由以色列公司 Better Place 于2007年首次商业化尝试，因电池标准碎片化于2013年宣告失败。宁德时代于2021年推出"巧克力换电块"，率先在商用车领域建立标准，为换电商业化提供技术前提。

当前换电主要集中于重卡市场。重卡在矿区、港口、长途走廊等场景对补能时效要求极高，换电（3-5分钟）相较快充（1-2小时）时间经济价值显著，形成清晰的付费逻辑。时代电服作为宁德时代旗下独立运营平台，目前是商用车换电赛道的领导者，换电站主要布局内蒙古、山西、陕西矿区走廊。初稿收到以后要做三件事：第一，核实所有数字——AI给出的数字不管看起来多权威都要查一遍；第二，把自己对合作方向的判断加进去，这个AI不知道你们团队跟这家公司的具体关系；第三，结尾加上你们对这个客户的定位和下一步打算，这是给领导看的。因为你更了解你的领导风格，我建议这一步就不要调教AI去做，它做出来的你大概率要很耗精力去改。初稿15分钟，审改40分钟，大概1小时。第四步：做PPT（约1.5小时）PPT这步不用从头做，先让AI把内容框架出来，再用工具自动排版。内容框架的Prompt：基于上面的材料，帮我做一个拜访前内部PPT的内容框架。

· 未来12个月扩张计划和资金节奏？

· 投资人关系是宁德时代统筹还是独立对接？框架出来以后，排版用Gamma（有科学上网，gamma.app，粘贴框架内容自动生成PPT，商务风格稳定）；国内没有科学上网就用WPS AI，或者直接套公司模板手动填。PPT这步，内容填充15分钟，排版和格式调整大概1-1.5小时，主要时间花在对齐风格和核实数字上。有一点要说清楚：这个PPT是内部用的，不是给客户看的。给客户看的正式材料，AI只能出框架和初稿，后续的数字核实、判断加工、格式规范，都还是要靠你自己。AI是第一遍，你是最后一遍，这个顺序一定不要反过来。番外：准备"水下信息"问题清单这一步是我觉得最被低估的，也是最能拉开差距的。大多数人做完材料就去见面了。但见面之前，真正值钱的是你知道哪些问题是对方不会主动说的。能决定这个项目走不走得通的那些信息，一般都不在公开资料里。
我们习惯叫这类信息"水下信息"。换电站的单站经济模型，宁德时代对时代电服的独立性有什么约束，竞争对手最近有没有什么动作，这些东西要么是商业机密，要么是行业里心照不宣的东西，你在研报里找不到的。但AI能帮你想到：哪些问题是你应该问的，以及怎么问才不会显得冒犯。Prompt是这样的：关于时代电服和换电行业，有哪些重要信息一般不在公开渠道？

如果我明天要去见他们的BD总监，给我两类问题：
① 适合直接问的（回答能帮我判断合作方向）
② 需要换个方式问的（太直接可能冒犯）AI出来的问题清单是这样的——【直接问的问题】你们现在在哪几个省重点布局，接下来12个月的扩张计划往哪个方向走？
（帮你判断融资窗口期在哪——如果扩张期还在加速，股权融资需求就是真实的）换电站的单站经济模型，你们内部一般怎么算，大概多少换电次数能回本？
（帮你判断Pre-IPO时机，以及核实AI给你的那个经济模型是否靠谱）你们现在宁德时代之外，有没有引入外部机构投资人？
（帮你判断是否已有主承销商在跟进，还是这块还是空的）有没有出海计划，比如东南亚或者中东的矿区或港口？
（帮你判断境外融资需求是否成立）【需要换个方式问的问题】不要直接问"你们和某某车企是独家合作"——太直接，可能引发敏感。换成：
"你们的标准接入体系是开放式接入，还是有优先合作梯度？"不要直接问"你们最近在跟哪家投行谈"——换成：
"你们资金面的节奏怎么样，近期有没有什么安排？"宁德时代对你们的独立性有什么限制——这个直接别问。换成：
"你们在投资人关系这块，现在是宁德时代统筹，还是你们自己有单独对接渠道？"这份问题清单，是你和你的领导在会议室里的一种气质、技巧和底气，比Word、PPT材料本身值钱得多。
AI工具方面这边，有没有科学上网都有对应方案有科学上网：Perplexity搜索，Claude整合和写材料，Gamma做PPT。没有：秘塔AI搜索，Kimi整合和写材料，WPS AI做PPT或直接套公司模板。效果差距见仁见智，主要是工具熟悉程度的问题。顺便说一下，这个流程里每一步AI给你的内容，都不是直接用的版本。搜集信息那步，你要核实数字；整合那步，你要加自己的判断；Word初稿，你要加团队定位；PPT，你要对齐公司模板和风格。AI是加速器，不是替代品。不过"加速"这件事本身就已经很值钱了——以前你花一个晚上，现在你花一个下午，节省下来的时间是你自己的。这个流程熟悉之后，可以大概计算一下，从收到领导微信到材料发出去，不到四个小时。傍晚接到任务，九点不到，就可以把材料发出去，然后该干嘛就干嘛去。以前这种事大概要花多少时间呢？我估计工作超过八九年的投行人应该都相信我说的：必然通宵。现在再为了这种事通宵，绝对算是没苦硬吃了……
